# U-REACH

**Umbrella-Review, Evaluation, Assessment, and Communication-Hub**

U-REACH is a flexible methodology to share scientific digital data about health. It bridges the gap between scientific evidence and the real world by combining rigorous systematic evidence synthesis with interactive online platforms.
